*Requirements:*
Preferably PHP5 and MySQL on an Apache server with mod_rewrite enabled

*Installation:*
1. Download/fork/fetch the full repository from GitHub
2. Create a MySQL database from scheme: db.sql
3. Change the following value in DB to reflect current site URL
	Table: setting
	Entry: config_url
4. Change configuration settings in config.php and admin/config.php (if needed)
